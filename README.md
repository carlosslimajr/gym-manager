# gym-manager

## Build with
 - Express JS 4.17
 - NunjucksJS 3.2.1
 - Method-overrideJS 3.0.0
 - PostgreeSQL

## Install
 1.  Be sure to have NodeJS v12.16.1 or later
 2. Clone the project and go to the directory root
 3. Execute npm install

 ## Run
 1. Execute npm start
